***HTML***

1. **What is web?**
   - The web refers to the World Wide Web, a system of interconnected documents and resources accessible via the internet.
   - It's a global information space where documents and resources are identified by Uniform Resource Locators (URLs).

2. **Difference between website and web app?**
   - A website primarily displays static content and provides information.
   - A web app is interactive, allowing users to perform tasks, manipulate data, and often requires user authentication.

3. **What is frontend?**
   - Frontend refers to the client-side of web development, involving the presentation and user interface of a website or web application.
   - It includes technologies like HTML, CSS, and JavaScript.

4. **What is backend?**
   - Backend is the server-side of web development responsible for handling data, business logic, and server operations.
   - It includes server-side languages like Node.js, Python, Ruby, and databases.

5. **What is a database?**
   - A database is a structured collection of data that can be easily accessed, managed, and updated.
   - It organizes and stores information for efficient retrieval and manipulation.

6. **What is NoSQL and SQL?**
   - SQL (Structured Query Language) is used for managing and manipulating structured data in relational databases.
   - NoSQL databases are non-relational, providing flexible and scalable data storage solutions.

7. **What is MongoDB and PostgreSQL?**
   - MongoDB is a NoSQL database that stores data in flexible, JSON-like documents.
   - PostgreSQL is a powerful open-source relational database management system (RDBMS).

8. **What is MERN stack?**
   - MERN stack comprises MongoDB, Express.js, React, and Node.js, forming a full-stack JavaScript framework for building web applications.
   - It allows end-to-end JavaScript development.

9. **How to make money using web dev skill?**
   - Freelancing, building and selling web applications, or offering web development services are common ways.
   - Creating and monetizing personal websites or blogs, or working as a web developer for companies.

10. **What about freelancing, applying to startups, and building own product?**
    - Freelancing offers flexibility and diverse projects; startups provide a dynamic work environment, and building your product allows creative control.
    - Each option has its pros and cons, depending on personal goals and preferences.

11. **What is a network?**
    - A network is a collection of computers and devices connected to share resources and information.
    - Networks enable communication and data transfer between connected devices.

12. **What is LAN, MAN, WAN?**
    - LAN (Local Area Network) covers a small geographic area like an office.
    - MAN (Metropolitan Area Network) spans a city, connecting multiple LANs.
    - WAN (Wide Area Network) covers a large geographical area, often connecting across cities or countries.

13. **What is the internet?**
    - The internet is a global network of interconnected networks that share information using standard protocols.
    - It provides a vast array of services, including the World Wide Web, email, and file sharing.

14. **What is WWW?**
    - WWW stands for World Wide Web, a system of linked hypertext documents accessible via the internet.
    - It is often used interchangeably with the term "web."

15. **What is IP address?**
    - An IP address is a numerical label assigned to each device connected to a computer network.
    - It serves two main purposes: host or network interface identification and location addressing.

16. **What is a browser actually? State different names of browsers?**
    - A browser is a software application that enables users to access and navigate the World Wide Web.
    - Examples of browsers include Chrome, Firefox, Safari, Edge, and Opera.

17. **What is a client and server?**
    - A client is a device or application that requests services or resources from a server.
    - A server is a computer or system that provides resources or services to clients over a network.

18. **What is DNS and domain?**
    - DNS (Domain Name System) translates human-readable domain names into IP addresses.
    - A domain is a human-readable name assigned to a specific IP address on the internet.

19. **What is ISP?**
    - ISP (Internet Service Provider) is a company that provides internet access to customers.
    - Examples include Comcast, AT&T, and Verizon.

20. **What is OSI model and its layers with a short summary?**
    - The OSI model (Open Systems Interconnection) is a conceptual framework with seven layers: Physical, Data Link, Network, Transport, Session, Presentation, and Application.
    - Each layer serves a specific function, facilitating communication between devices on a network.

21. **What are protocols? State different types of protocols?**
    - Protocols are rules defining communication between devices on a network.
    - Different types include HTTP/HTTPS (web communication), FTP (file transfer), SMTP/IMAP/POP3 (email), and TCP/IP (internet communication).

22. **How does a packet travel? Give the entire flow of transferring a packet?**
    - Data is broken into packets, routed through a network, and reconstructed at the destination.
    - The process involves encapsulation, addressing, routing, and de-encapsulation.

23. **What is HTML?**
    - HTML (Hypertext Markup Language) is the standard markup language for creating web pages.
    - It structures content using tags and elements.

24. **Structure of HTML code?**
    - HTML code consists of elements enclosed in tags, forming a hierarchical structure.
    - Elements can have attributes providing additional information.

25. **Is HTML, head, and body tag the structure of HTML?**
    - Yes, HTML, head, and body tags collectively form the basic structure of an HTML document.
    - HTML contains metadata, head contains title and links, and body contains the content.

26. **What is metadata, and how is it used for SEO purposes?**
    - Metadata provides information about the HTML document, including title, character set, and description.
    - It influences how a page is displayed in search engine results, impacting SEO.

27. **What is an element and tag, give the difference between them?**
    - An element is a complete set of tags and content, while a tag is the opening or closing part of an element.
    - Example: `<p>This is an <strong>element</strong>.</p>`

28. **Key difference between HTML4 and HTML5?**
    - HTML5 introduces new semantic elements, native support for audio and video, and enhanced form controls.
    - It offers improved compatibility with mobile devices and better structure.

29. **Importance of indentation?**
    - Indentation improves code readability and helps identify the structure of HTML elements.
    - It is a best practice for maintaining clean and organized code.

30. **What are comments in HTML?**
    - Comments in HTML are <!-- like this --> and are used to add notes or explanations in the code.
    - They are ignored by the browser and are for the developer's reference.

31. **What are tags in HTML? Give types of tags in HTML?**
    - Tags are keywords enclosed in angle brackets and define HTML elements.
    -

 Types include opening `<tag>` and closing `</tag>` tags, self-closing tags like `<img />`, and block vs. inline tags.

32. **What are semantic tags? List down different tags?**
    - Semantic tags provide meaning to the content and help structure a page.
    - Examples include `<header>`, `<article>`, `<footer>`, `<nav>`, and `<section>`.

33. **What are structure tags? List down different tags?**
    - Structure tags define the basic structure of an HTML document.
    - Examples include `<html>`, `<head>`, `<body>`, `<title>`, and `<meta>`.

34. **What are formatting tags? List down different tags?**
    - Formatting tags define the appearance of text and elements.
    - Examples include `<b>`, `<i>`, `<u>`, `<strong>`, and `<em>`.

35. **What are self-closing tags? List down different tags?**
    - Self-closing tags don't require a closing tag and end with a slash before the closing angle bracket.
    - Examples include `<img />`, `<br />`, and `<input />`.

36. **What is block and inline elements? Give the difference between them?**
    - Block elements create a new block-level box, while inline elements flow within the content.
    - Block elements start on a new line and take the full width, whereas inline elements only take necessary width.

37. **What are Emmets exactly? Give me different Emmets?**
    - Emmet is a toolkit for web developers to increase HTML and CSS coding efficiency.
    - Examples: `div>ul>li*3>a`, expands to a div containing an unordered list with three list items, each containing an anchor.

38. **What is an abbreviation? Tell me with tag symbol, and etc.?**
    - Abbreviations are shortened forms of words, and in HTML, they are represented using the `<abbr>` tag.
    - Example: `<abbr title="Hypertext Markup Language">HTML</abbr>`.

39. **What is a quotation and citation with an example?**
    - Quotation is used for short, inline quotes and is represented by the `<q>` tag.
    - Citation is used to reference the source of content and is represented by the `<cite>` tag.

40. **What is marquee?**
    - The `<marquee>` tag was used for creating a scrolling or moving text effect.
    - It's deprecated in HTML5, and CSS or JavaScript is recommended for similar effects.

41. **What is iframe?**
    - The `<iframe>` tag is used to embed another document or webpage within the current document.
    - It's commonly used for embedding videos, maps, or other external content.

42. **What are image, audio, and video tags?**
    - `<img>` is used for embedding images, `<audio>` for audio files, and `<video>` for video files.
    - They allow multimedia content to be embedded directly into the HTML document.

43. **What is the embed tag, and how can we use it?**
    - The `<embed>` tag is used to embed external content, like plugins or multimedia.
    - Example: `<embed src="example.swf" type="application/x-shockwave-flash">`.

44. **How can we embed PDF, YouTube video, and Google Maps?**
    - PDF: Use `<embed>` or `<object>` tags or an `<iframe>` with a PDF URL.
    - YouTube Video: Use an `<iframe>` with the YouTube video URL.
    - Google Maps: Use an `<iframe>` with the Google Maps URL.

45. **What are HTML tables?**
    - HTML tables are used to organize data into rows and columns.
    - They consist of the `<table>`, `<tr>` (table row), `<td>` (table data), and `<th>` (table header) elements.

46. **Table properties of HTML elements?**
    - Table properties include `<caption>` for table captions, `<col>` for column properties, and `<colgroup>` for column groups.
    - Additionally, attributes like `border`, `cellpadding`, and `cellspacing` affect table appearance.

47. **What are advanced attributes of HTML tables? Rowspan and colspan.**
    - `rowspan`: Specifies the number of rows a table cell should span.
    - `colspan`: Specifies the number of columns a table cell should span.

48. **What are HTML forms?**
    - HTML forms are used to collect user input, and they consist of various form elements.
    - Forms are created using the `<form>` tag.

49. **What is the input tag in HTML forms, and which are those?**
    - The `<input>` tag is used to create various form controls such as text fields, buttons, checkboxes, and radio buttons.
    - Examples: `<input type="text">`, `<input type="submit">`, `<input type="checkbox">`.

50. **What is a textarea?**
    - The `<textarea>` tag is used for multiline text input in forms.
    - Example: `<textarea rows="4" cols="50">Enter text here...</textarea>`.

51. **What is a label?**
    - The `<label>` tag is used to associate text with a form control.
    - It improves accessibility and user experience.

52. **What is a name in HTML forms?**
    - The `name` attribute is used to identify form controls when submitting data to the server.
    - It helps in processing form data on the server side.

53. **What is a placeholder?**
    - The `placeholder` attribute provides a hint or example text within a form control.
    - It disappears when the user starts typing.

54. **What is a dropdown?**
    - A dropdown is a UI element that allows users to select from a list of options.
    - It's created using the `<select>` and `<option>` tags.

55. **What is a button, fieldset, and legend in HTML forms?**
    - `<button>` is used for clickable buttons within forms.
    - `<fieldset>` groups related form elements, and `<legend>` provides a title for the fieldset.
